Device setup guides — Fire TV, Android, smart TV & more

Step-by-step Monster Streams onboarding for popular devices. Same-day setup is typical when instructions are followed on a compatible device.

Firestick / Fire TV / Fire Cube Setup
▼

Install Downloader from Fire TV Search (search Downloader → install).

Sideloading: Settings → My Fire TV → tap My Fire TV seven times if Developer Options is hidden → Developer Options → allow Unknown Apps for Downloader.

Open Downloader. If Settings appears first: open Settings, turn Download on, go back and continue.

Install one of the following (Monster TiviMate or Monster Smarters v5). In Downloader enter the code in the URL box and press Go, or paste the direct URL:

Option A — Monster TiviMate

Downloader code: 5135389
Direct URL: https://tinyurl.com/montivi

Option B — Monster Smarters v5

Downloader code: 6415914
Direct URL: https://tinyurl.com/monsm5

Install and open the app:

If you chose Monster TiviMate: add your Xtream playlist using the portal URL and credentials Monster Streams supplied.

If you chose Monster Smarters v5: tap Monster TV or TV, then enter Username and Password.

From the Fire TV home screen Search, install VLC Player.

Existing customers: uninstall the old Monster Smarters v4 build (Settings → Applications → Manage Apps) and delete stale APK downloads from Downloader storage.

Updates: Settings → Playlist → Update all playlists twice after install—or open Live TV / Movies / Series so each section pulls fresh catalog data.

VPN: You may need a VPN during peak sports. Surfshark suggestion: France (Paris), Germany (Frankfurt), or Multi-Host France → Sweden. Reset your Wi-Fi router once before configuring.

Factory-reset help (Amazon/Fire OS): watch the walkthrough video. If you wipe the device, set VPN to France (Marseille) before reinstalling APKs.

Android Box / Android TV Setup
▼

Install Downloader from Google Play Store, then locate it anytime via Home → Settings → Applications → Manage applications.

Open Downloader and install one of the APKs below (Monster TiviMate or Monster Smarters v5).

Option A — Monster TiviMate

Downloader code: 5135389
Direct URL: https://tinyurl.com/montivi

Option B — Monster Smarters v5

Downloader code: 6415914
Direct URL: https://tinyurl.com/monsm5

Install and launch the app:

TiviMate users: add your Xtream playlist with the server URL/username/password provided by Monster Streams.

Monster Smarters v5 users: select Monster TV, Monster Phone, or TV per the banner, then sign in.

Install VLC Player from Google Play.

Existing customers: remove Monster Smarters v4 (Settings → Applications → Manage Apps) and clear old Downloader downloads if you no longer need them.

Updates: Settings → Playlist → Update all playlists twice, or open each content section to refresh.

VPN: Surfshark suggestion for sports: France (Marseille), Germany (Frankfurt), or MultiHop France → Sweden. Reset your Wi-Fi router once before configuring.

Factory-reset help: Android Box video · Android TV video. Set VPN to France (Marseille) before re-entering download codes.
